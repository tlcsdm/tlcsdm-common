## tlcsdm-common
常用工具类封装

1.4.0开始基于java11 构建  
java1.8可以使用 1.3.4

## Maven
```
<dependency>
    <groupId>com.tlcsdm</groupId>
    <artifactId>tlcsdm-common</artifactId>
    <version>1.4.0</version>
</dependency>
```
